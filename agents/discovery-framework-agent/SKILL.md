# Discovery Framework Agent

Extract insights from client discovery calls and transcripts.

## Operations
- Parse transcripts (PDF, text, JSON)
- Identify pain points & opportunities
- Extract tech stack details
- Compare discovery patterns across clients
- Generate discovery questionnaires
- Output JSON for downstream agents

## Scripts
- `extract-patterns.js` — Parse transcript → JSON
- `compare-clients.js` — Compare discovery patterns
- `generate-questionnaire.js` — Build discovery Q&A
