# Data Model Agent

Design and normalize database schemas.

## Operations
- Design entity models
- Normalize schemas (1NF → 3NF)
- Map client data to standard model
- Generate SQL schema
- Document relationships
- Design migrations

## Scripts
- `design-schema.js` — Create entity model
- `normalize.js` — Normalize to 3NF
- `generate-sql.js` — Output CREATE TABLE statements
