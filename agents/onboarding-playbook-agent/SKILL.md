# Onboarding Playbook Agent

Create structured onboarding timelines and phase plans.

## Operations
- Generate 6-week phase breakdown
- Define deliverables per phase
- Identify decision gates
- Create SOW template
- Build resource allocation plan
- Design success metrics

## Scripts
- `create-timeline.js` — 6-week plan
- `define-phases.js` — Phase breakdown
- `generate-sow.js` — Statement of Work
