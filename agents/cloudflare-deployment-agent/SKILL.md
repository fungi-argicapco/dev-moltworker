# Cloudflare Deployment Agent

Deploy and manage Cloudflare Workers, Pages, and services.

## Operations
- Deploy Workers code
- Deploy Pages sites
- Manage versions & rollbacks
- Trigger deployments
- Monitor deployment status

## Scripts
- `deploy-worker.js` — Deploy Worker
- `deploy-pages.js` — Deploy Pages site
- `rollback.js` — Rollback to previous version
